# Deck « Prof » — images des figures

Déposez ici les photos du professeur pour remplacer les figures :

- `jack.png`  → Valet (J)
- `queen.png` → Dame (Q)
- `king.png`  → Roi (K)

Formats : PNG ou JPG, idéalement portrait (ratio ~3:4), au moins 300×400 px.

Tant qu'un fichier est absent, la carte correspondante retombe automatiquement
sur le rendu par défaut (grande lettre + enseigne) — le rang et l'enseigne
restent toujours visibles dans les coins, quel que soit le deck.

Pour ajouter d'autres decks, voir `client/src/app/blackjack/decks.ts`.
